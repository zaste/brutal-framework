BRUTAL V4 - Lifecycle Hooks Test

Lifecycle Hook Phases

This test demonstrates all component lifecycle hooks:

  • beforeCreate - Before component initialization
  • created - After component initialization
  • beforeMount - Before connecting to DOM
  • mounted - After connecting to DOM
  • beforeUpdate - Before attribute changes or re-render
  • updated - After attribute changes or re-render
  • beforeUnmount - Before disconnecting from DOM
  • unmounted - After disconnecting from DOM
  • errorCaptured - When an error occurs
