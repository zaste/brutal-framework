Symbiotic Testing Philosophy

In BRUTAL V4, tests aren't external validators - they ARE components that test themselves. This demo shows the render scheduler test running as a visual component.

Key insight: The test uses BrutalState, BrutalEvents, and Shadow DOM - the same systems it's testing. True dogfooding.
