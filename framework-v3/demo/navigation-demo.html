Customization

The NavigationBar component supports various customization options:

  • sticky: Enable/disable sticky positioning
  • transparent: Enable transparent background
  • brand: Set the brand text
  • logo: Set the logo image URL
  • breakpoint: Set mobile breakpoint (default: 768px)
  • items: Configure navigation items via JSON

Performance

The component uses several performance optimizations:

  • Throttled scroll events (60fps)
  • Debounced resize handling
  • GPU-accelerated animations
  • Passive event listeners
  • RequestAnimationFrame for smooth updates
